# E-commerce FastAPI Workshop
Git/GitHub Workshop - Học Git workflow thông qua xây dựng E-commerce API
## Setup
\`\`\`bash
# Install dependencies
pip install -r requirements.txt
# Run server
uvicorn app.main:app --reload
\`\`\`
## API Documentation
Once running, visit:
- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc
